RGB Colour Picker

RGB Colour Picker is a versatile tool for exploring and converting colours. Select a colour to instantly display its HEX, RGB, CMYK, HSV, and HSL values. Entering a value in any field automatically generates the corresponding equivalents in all other formats, along with the exact point on the colour field.
